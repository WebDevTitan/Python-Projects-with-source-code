# A Bootstrap Theme Blog Powered By Flask 

A simple and full responsive Bootstrap blog.

##  Technologies

### Frontend
* HTML 5
* CSS 3
* JavaScript
* Bootstrap 

### Backend
* Flask 


## Installation

1. First clone or download the files.
2. Install Flask framework
   ```bash
   pip install Flask
   ```
3. Install requests module
   ```bash
   pip install requests
   ```
4. Run the app using 
    ```bash
    flask run
    ```
